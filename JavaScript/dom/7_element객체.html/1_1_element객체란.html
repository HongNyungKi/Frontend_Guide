이상, 변수 t 가 가르키는 객체는 htmllielement가 된다. 이것은 li가 갖고있는 특성을 반영하기 위한 객체이다. 그리고 모든 html객체들은 htmlelement의 자식이다. htmlelemnet는 모든 html 태그들이 공통적으로 갖고있는 속성을 정의하는 객체이다. 대표적으로 style속성이 있다.(css를 제어하는) htmlelemnet도 elemnet객체 라는 부모를 갖고있다. (문서상의 모든 elemnet의 기능을 정의하는) htmlelement객체 와 elemnet객체 부모 자식 관계로 나누는 이유는 dom(document object model)이 반드시 html만을 제어하기 위함이 아니기 때문이다. (HTML이나 XML, SVG, XUL과 같이 마크업 형태의 언어를 제어하기 위한 규격이기 때문) 각각의 언어들의 모든 element들에게 기능을 적용하기 위해 element객체가 있다. element객체가 갖고 있는 api는 크게 식별자api / 조회api / 속성api 등이 있다.
