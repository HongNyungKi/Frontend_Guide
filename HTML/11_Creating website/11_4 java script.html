java script이란 무엇인가?!

우리가 매일 접속하는 웹사이트는 크게 3가지 요소로 구성된다. ‘HTMLHyper Text Markup Language’, ‘CSSCascading Style Sheets’, ‘자바스크립트Javascript’다. HTML은 웹페이지의 큰 뼈대를 제공하고, CSS는 색깔이나 글씨체와 같은 디자인 요소를 관리한다. 자바스크립트는 크로스 플랫폼cross platform, 객체지향 스크립트 언어로 웹페이지의 동작을 담당한다. 예를 들어 자바스크립트를 이용하면 ‘버튼을 클릭하면 밑에 날짜를 보여줘’라는 식의 명령을 내릴 수 있다. 네이버 지식백과 자바스크립트 Javascript - 웹을 풍부하게 만들어주는 작고 가벼운 언어 용어로 보는 IT, 이지현
