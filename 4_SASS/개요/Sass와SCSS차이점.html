SCSS가 SASS의 상위 집합이다. 즉, 2번과 같은 이유로 SASS에서 SCSS를 도입한 것이다. 1.SASS .list width: 100px float: left li color: red background: url("./image.jpg") &:last-child margin-right: -10px 2. SCSS .list { width: 100px; float: left; li { color: red; background: url("./image.jpg"); &:last-child { margin-right: -10px; } } } 1. SCSS @mixin border-radius($radius) { -webkit-border-radius: $radius; -moz-border-radius: $radius; -ms-border-radius: $radius; border-radius: $radius; } .box { @include border-radius(10px); }
