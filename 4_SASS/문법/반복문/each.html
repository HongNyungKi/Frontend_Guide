@each는 list 와 map 데이터를 반복할 때 사용한다. js의 for in 문과 유사하다. list는 js의 배열과 map은 js의 객체와 '유사'하다. @each $변수 in 데이터 { 반복내용 } 2. list데이터 반복하기 $fruits: (apple, orange, banana, mango); .fruits{ @each $fruit in $fruits{ li.#{$fruit}{ background: url("image/#{$fruit}.png"); } } } .fruits li.apple { background: url("image/apple.png"); } .fruits li.orange { background: url("image/orange.png"); } .fruits li.banana { background: url("image/banana.png"); } .fruits li.mango { background: url("image/mango.png"); }
