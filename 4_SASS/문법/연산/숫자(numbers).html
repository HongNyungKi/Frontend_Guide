1. 상대적 단위 연산 : 일반적으로 절댓값을 나타내는 px 단위로 연산을 한다. 하지만 상대적 단위(%,em,yw 등)의 연산의 경우 CSS calc()로 연산해야 한다. ex) width: 50% -20px; //단위 모순 에러 width: calc(50% -20px); //연산 가능 2. 나누기 연산의 주의사항 : CSS는 속성 값의 숫자를 분리하는 방법으로 / 를 혀용하기 때문에 / 가 나누기 연산으로 사용되지 않을 수 있다. 예를 들어, font: 16px / 22px serif; 같은 경우 font-size:16px 과 line-height: 22px 의 속성값 분리를 위해서 / 를 사용한다. ex) 나누기 연산자만 연산되지 않고 그대로 컴파일 되는 경우 .div{ width: 20px + 20px; height: 40px -10px; font-size: 10px*2; margin:30px/2; } .div { width: 40px; height: 40px -10px; font-size: 20px; margin: 30px/2; //error } 3. 나누기 연산의 조건 - 값 또는 그 일부가 변수에 저장되거나 함수에 의해 반환되는 경우 - 값이 ()로 묶여있는 경우 - 값이 다른 산술 표현식의 일부로 사용되는 경우 .div{ $x:100px; width:$x/2; //변수에 저장된 값을 나누기 height:(100px/2); // 괄호로 묶어서 나누기 font-sixe:10px+12px/3 //다른 연산과 같이 사용하여 나누기 }
