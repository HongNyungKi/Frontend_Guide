SASS mixins는 스타일 시트 전체에서 재사용 할 CSS 선언 그룹을 정의하는 기능이다. 약간의 mixin으로 다양한 스타일을 만들어 낼 수 있다. mixin은 선언하기(@mixin)와 포함하기(@include)로 나뉜다. 만들어서(mixin),사용(include)하는 것이다. @mixin 믹스인이름 (밖에서 받아올 내용){스타일;} 단, 밖에서 들어오는 값이 없을 경우 ()생략 가능하다. SASS에서 사용할 경우 @ 대신 = 를 쓰고 {}와 ; 생략하면 된다. @include대신에는 + 를 사용한다. @mixin size($w, $h){ width:$w; height:$h; } .box1{ @include size(100px, 100px) } .box2{ @include size(100px, 100px) } .box3{ @include size(100px, 100px) } .box1 { width: 100px; height: 100px; } .box2 { width: 100px; height: 100px; } .box3 { width: 100px; height: 100px; } @mixin size($w, $h)에서 밖에서 들어오는 값이 없을 경우 @mixin size($50, $50){ width:$w; height:$h; } 처럼 기본값을 적용하여 사용할 수 있다. @mixin size($50, $50){ width:$w; height:$h; } .box1{ @include size(200px,300px) } .box{ @include size($h:300px) } @mixin large-text { font: { size: 22px; weight: bold; family: sans-serif; } color: orange; &::after { content: "!!"; } span.icon { background: url("/images/icon.png"); } }
