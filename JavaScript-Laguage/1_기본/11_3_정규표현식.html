정규표현식 패턴을 만들 때 옵션을 설정할 수 있다. 옵션에 따라서 검출되는 데이터가 달라진다. 1. i를 붙이면 대소문자를 구분하지 않는다. var xi = /a/; "Abcde".match(xi); var oi = /a/i; "Abcde".match(oi); 결과 ["A"] 2. g를 붙이면 모든 결과를 리턴한다. var xg = /a/; "abcdea".match(xg); var og = /a/g; "abcdea".match(og); 3. 옵션들을 같이 사용 할 수도 있다. var ig = /a/ig "AabcdAa".match(ig);
