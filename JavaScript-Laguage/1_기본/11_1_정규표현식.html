정규표현식(rgular expression)이란 문자열에서 특정한 대상의 유무를 확인(test)하는 기능, 치환하는 기능, 추출하는 기능의 도구라 할 수 있다. 자바스크립트에서 정규표현식을 사용하는 방법을 배운다. 정규표현식은 두가지 단계로 이루어진다.(컴파일, 실행) -컴파일: 내가 필요한 자료 혹은 대상 즉, 패턴을 찾는 것이다. -실행: 내가 찾은 패턴에 대해서 작업을 하는 것. 컴파일은 검출하고자 하는 패턴을 만드는 일이다. 정규표현식을 컴파일해서 객체를 만들자. 이는 2가지가 있다. 1. 정규표현식 리터럴: var pattern = /a/; 2. 정규표현식 객체 생성자: var pattern = new RegExp('a'); 둘다 a라는 문자의 정보를 pattern이라는 변수에 저장한 것이다. 정규표현식을 컴파일해서 객체를 만들었다면 이제 문자열에서 원하는 문자를 찾아내야한다. 1. exec라는 정규표현식에 소속되어있는 메소드 사용시(추출에 용이) : console.log(pattern.exec('abcdef')); 실행결과는 문자열 a를 값으로 하는 배열을 리턴한다. ['a'] : console.log(pattern.exec('bcdfg')); 인자 'bcdfg'에는 a가 없기 때문에 null을 리턴한다. 2. text는 라는 정규표현식에 소속되어있는 메소드 사용시(존재유무 테스트에 용이) : console.log(pattern.test('abcdef')); 리턴값으로 boolean이 나온다. true : console.log(pattern.test('bcde'));
