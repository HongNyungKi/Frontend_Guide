@@@ 블럭 레벨 요소(Block level)와 인라인 요소(inline) @@@ 1. 블럭 요소 : 사용가능한 최대 가로 너비를 사용한다. 크기를 지정 할 수 있다. (width:100%, height:0;)로 시작 margin,padding의 상,하,좌,우 값이 모두 적용된다. layout을 다루는 용도로 활용 ex) div / h1 / p .... 2. 인라인 요소 : 필요한 만큼의 너비를 사용한다. 크기를 지정 할 수 없다. (width:0, height:0;)로 시작 코드 입력 시 줄바꿈을 하면 자동 띄어쓰기가 된다. margin,padding의 좌,우 값만 적용이 된다. text를 다루는 용도로 활용. ex) span / img ... * css의 속성인 display로 요소의 종류를 바꿀수 있다. ex) div{diplay: inline;} span{display: block;}
